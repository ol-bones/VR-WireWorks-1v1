# A VRChat 1v1 dot boxes game

How to modify:
1. go to https://github.com/ol-bones/VR-WireWorks-1v1/releases/tag/1
2. download the zip
3. download VRChat Creator Companion
4. Unzip the zip
5. Import the folder into VCC
6. Drag the scene prefab into the hierachy
